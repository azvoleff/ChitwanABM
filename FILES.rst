===============================================================================
Overview of ChitwanABM Files
===============================================================================


1) Main model code:
    • __init__.py - Defines ChitwanABM module (required for Python).
    • agents.py - Defines the agent classes used in the ChitwanABM.
    • batch_run.bat - Used to run a set of ABM runs of the same scenario.
    • calc_NBH_stats.R - Plotting code - used after running the model to plot 
    results at the neighborhood level.
    • data_preprocess.R - Used to produce initialization data for the model from 
    ICPSR restricted use datafiles (see REAMDME.txt for more details).
    • FILES.rst - This file, a reStructuredText format file listing the files that 
    make up the ChitwanABM.
